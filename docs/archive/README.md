# Archive

This directory keeps historical material for provenance. It is not part of the normal documentation path for using the repository.

## Included

- `academic-report/`: course-report drafts, LaTeX sources, exported figures, and rendering assets
- `experiments/`: per-round experiment summaries, comparison notes, and retained intermediate tables
- `source-assets/`: source images referenced by the archived report materials

## Usage

- Prefer `../guides/`, `../reports/`, and `../tables/` for normal repository use.
- Use this archive only when you need historical context or want to trace how a result was derived.
