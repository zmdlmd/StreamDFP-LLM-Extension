Selected SMART attributes

These columns are shown because they are common raw indicators for media, power, workload, and temperature evidence.

attribute typical meaning possible failure signal role in this demo
SMART 1 raw Read Error Rate. Vendor-specific raw counter related to low-level read behavior. May indicate unstable reads or media-related stress, but raw values require model-aware interpretation. Shown as a raw signal candidate; grouped as unknown in this summary because it is not the strongest rule-backed cause.
SMART 5 raw Reallocated Sectors Count. Counts sectors remapped by the drive. Often associated with media degradation when it increases. Included as a standard media-risk reference; remains zero in the sampled rows.
SMART 9 raw Power-On Hours. Approximate accumulated operating time. Reflects workload or age exposure rather than a direct failure by itself. Contributes weak workload evidence in the Phase 1 rule summary.
SMART 187 raw Reported Uncorrectable Errors. Counts errors the drive reports as uncorrectable. Can indicate reliability stress or latent media problems. Used in the signal sketch as an additional error-related trajectory.
SMART 193 raw Load/Unload Cycle Count. Mechanical load/unload activity counter. May reflect power-state or mechanical activity stress. Mapped to power evidence in the rule layer with severity 1.00.
SMART 194 raw Temperature. Raw operating temperature reading. High or abnormal temperature behavior may indicate thermal stress. Main temperature evidence; drives the top rule score temperature=0.552.

How RULE_SCORE is derived

Rule scores are computed before the LLM call from model-specific feature rules.

feature severity = clip(|current - baseline| / (2 * threshold))
contribution = weight * severity * reliability * state * recency
group score = top-k contribution sum normalized by available signals
Rule profile ST3000DM001 inherits the HDD profile. Features are mapped to groups such as temperature, power, media, interface, and workload.

Prediction guard temperature is selected because it is above the low-score threshold and its margin over power is large: 0.552 - 0.035 = 0.517.

Rule evidence inside Phase 1

The summary keeps the rule calculation visible instead of hiding it inside the prompt.

Feature ruleSMART_194 raw is mapped to the temperature group, while SMART_193 is mapped to the power group.
SeverityEach feature is compared with its early-window baseline and threshold, then clipped to a 0-1 severity value.
