## 结论

本次 `st31500541as` 的负增益更符合“模型/抽取链路退化”，而不是“子集波动”。

原因：

1. 对照是在同一 `pilot20k` 子集、同一日期切片、同一 Phase3 参数下完成的。
2. 旧快照在同子集上仍是正增益，`Qwen3.5` 在同子集上变成了显著负增益。
3. `Qwen3.5` 不只是某个组合失效，而是整张 Phase3 网格最优结果都没有翻正。
4. Phase2 的根因抽取分布明显塌缩，非 `unknown` 有效窗口从 `4614` 降到了 `1418`。

因此，当前更合理的判断是：

- `st31500541as` 的这次退化主要来自模型侧或抽取链路侧变化
- 不是由 `pilot20k` 子集抽样波动单独造成
